## 🚀 上传到 GitHub

### 1. 创建 GitHub 仓库

1. 登录 [GitHub](https://github.com)

3. 填写信息：

   - Description: `A simple Telegram forward and reply bot`

### 2. 上传文件

## 📝 重要提示

### ⚠️ 修改这些信息

在上传前，请**确保修改**以下内容：

```python
# tg_bot.py 中
ADMIN_ID = 7358821097              # ❌ 改成你的 ID
BOT_TOKEN = "你的Token"            # ❌ 改成你的 Token
```

### ✅ 推荐做法

使用环境变量而不是硬编码：

```python
import os
from dotenv import load_dotenv

load_dotenv()

ADMIN_ID = int(os.getenv('ADMIN_ID'))
BOT_TOKEN = os.getenv('BOT_TOKEN')
```

然后在本地使用 `.env` 文件（记得加入 `.gitignore`）
